## Troubleshooting

### "I don't see Add to Home Screen option"
- Make sure you're using Safari (not Chrome)
- Update to latest iOS version (iOS 15+ recommended)
- Try restarting Safari

### "App keeps opening in Safari"
- Make sure you tapped "Add to Home Screen" (not just bookmarked)
- Delete and reinstall from home screen

### "Icon looks wrong"
- Delete app from home screen
- Clear Safari cache: Settings > Safari > Clear History
- Reinstall using steps above

### "App won't install"
- Check your internet connection
- Make sure you have storage space on device
- Try restarting your iPhone/iPad
